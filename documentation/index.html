Introduction

AppLocker,an Android Native Application will help you to lock your application which you don't want to access without your permission. This application has feature of Pattern Lock and Password Lock. You can lock each application separately.   
 
Now you can restrict your applications to access without your permission. It will help you to prevent from Data loss from beginner users or childern.

Basic Setup for Android

To install and run any Android application we need following basic things.

  • Eclipse (editor to make changes in android application file)
  • Java Environment (to run application)
  • Android SDKs (to test android application with different environment or versions)

Eclipse

You can Download Eclipse from below URL according to your System configuration.

http://www.eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/junosr2

JAVA

You can install Java 6 from below URL:

http://www.oracle.com/technetwork/java/javase/downloads/java-archive-downloads-javase6-419409.html

Setup Java path

Only java installation is not enough, we need to set path for JAVA to run it. See the below URL.

http://docs.oracle.com/javase/tutorial/essential/environment/paths.html

Android Setup

After installing java and extracting eclipse, you need to follow the steps given in below URL to setup android environment:

 https://developers.google.com/eclipse/docs/install-eclipse-4.2

Workspace for Apps

Now in next step we need to setup Android Workspace. When we start Eclipse, it will ask to create Workspace which is use to store Android Project Location

To create/change workspace go to File Menu->Switch Workspace

 

 

If we have already created workspace, we can select it from list. And we want to create new, just click on Other and select your destination for it.

Import Android App

Now, the final and most important thing. How to import existing android application into workspace just follow the below steps:

  • File → Import → General → Existing Projects into Workspace → Next

     
  • Select root directory: /path/to/project


     
  • Projects → Select All
  • Check Copy projects into workspace


     
  • Click on Finish!!!

Cheers...! Now start changing in code and put your own things.
